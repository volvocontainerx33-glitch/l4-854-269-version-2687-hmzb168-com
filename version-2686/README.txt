高清日韩电影静态站点

生成内容：
- 首页：index.html
- 分类总览：categories.html
- 独立分类页：10 个
- 排行榜：ranking.html
- 搜索筛选页：search.html
- 影片详情页：2000 个
- HLS 播放器脚本：assets/player.js

封面图片说明：
所有影片封面与 Hero 轮播均按顺序引用网站根目录下的 1.jpg 到 150.jpg。
第 N 部影片对应图片编号为 ((N - 1) % 150) + 1。

播放源说明：
详情页播放器使用上传素材原 JS 中提取的 m3u8 播放源，并通过上传素材中的 HLS 模块初始化。

部署方式：
将本目录整体上传到静态服务器即可访问。若直接本地双击 HTML，部分浏览器可能会限制 ES Module 或 HLS 网络播放，建议使用任意静态服务器预览。
